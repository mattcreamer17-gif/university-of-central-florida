02

One layer up from AnyMAC.

Wang's AnyMAC treats multi-agent topology as iterative prediction. ICM treats the context and identity structure each agent carries before any topology assembles. The two are complementary and, as far as we can read, not yet in conversation with each other.

  1. L0Identity
    Who the agent is. Role, constraints, refusal posture.
  2. L1Domain
    Working knowledge of the field the agent operates inside.
  3. L2Procedure
    Contracts and stage definitions. What "done" looks like per phase.
  4. L3Factory
    Reusable patterns. The reference layer other agents compose against.
  5. L4Working
    Run-specific artifacts. The output the topology produces.

Repo (MIT license): github.com/RinDig/Interpretable-Context-Methodology-ICM-. 52-member practitioner community. U-shaped edit pattern observed across long runs.
